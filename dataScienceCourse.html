Epidemiology Track

This course introduces fundamental as well as applied data modelling techniques for epidemiology. The course topics include, but are not limited to: epidemiology and Infectious disease, exponential function, epidemic curve, disease transmission and severity measurement (basic reproduction number R0 and case fatality ratio CFR), robust estimation of various metrics, infectious diseases modelling (SIR model and IDEA model), model parameter estimation, and data visualization techniques using Tableau.

EP 101 – Introduction to Epidemiology

In this session , we will discuss what epidemiology is and why it is important. We will introduce some tools and metrics used in epidemiology to evaluate the transmissibility and serenity of an infectious disease like COVID-19. Topics covered: overview of the course; overview of epidemiology and infectious disease; outbreak, epidemic and pandemic; epidemiological triangle; epidemic curve; concept of exponential functions; disease transmissibility measures (attack rate, secondary attack rate, basic reproductive number R0); disease severity measures (case fatality risk, CFR; pandemic severity index; mortality rate; proportionate mortality); data visualization using Tableau Public: (plot epi curve).

EP 102 – Infectious Disease Modeling

In this session, we will introduce what mathematical models are and some basic but important mathematics concepts. We will discuss one of the most important mathematical models used in epidemiology, compartmental models and show how we can use this model to simulate the spread of a disease. Topics covered: introduction to disease modelling; definition of model; importance of a model; clinical iceberg model; robust estimate of parameter (censoring effect, CFR estimation); concept of rate of change; compartmental models: SIR model; relationship between SIR model and R0; compartmental models: SIS model.


EP 203 – Advanced Mathematical Models and Case Studies

In this session, we will discuss two more mathematical models used in epidemiology. The first one is the SEIR model, which is a compartmental model, and the second one is the IDEA model. We will explore what they are and how they are applied to analyze the spread of a disease. Topics covered: review compartment model; compartmental models: SEIR model; other methodology: IDEA model; case study and simulation: IDEA model; dependence on R0 and a discounting factor d.

EP 204 – Parameter Estimation Techniques

In this session, we will introduce some statistical quantities and tools to help us in parameter estimation. In data science, most of the time we need to estimate the parameters of a mathematical model. We will discuss how we can apply statistics to find the best parameters of a model. Topics covered: probability distribution; maximum likelihood estimation; Monte Carlo simulation; Root Mean Squared Error minimization.




Julia Programming Track

Data Science is a field of study on extracting meaning from structured or unstructured data by analysis of data with different processes and techniques. The invention of data science changed the paradigm of making decisions from intuition to data-driven. It brings many benefits to businesses. Julia is a programming language that is optimized for scientific and numerical computing. The features of Julia include easy to learn and high performance, such that it is suitable for data science. In this course, we will focus on Julia programming skills and how to develop data science applications based on Julia. The objectives of this program are: to provide students basic knowledge on Julia programming; to introduce programming techniques necessary for developing data science applications.

JU 201 Introduction to Julia

In this session, we will be looking at Julia from basics. We will set up the environment for running Julia and introduce the basic concept of programming and practice the concept in Julia. This course will cover: Overview of the course; Introduction to Programming; Overview of Julia; Setup the environment of Julia; Basic Julia language elements; Introduction to Function in Julia.

JU 202 Introduction to Control Flow in Julia

In this session, we will focus on the concept of control flow. It will include branching and looping, and practice the concept in Julia by if-statement, while-statement and for-statement. This course will cover: Introduction to Control Flow; Control Flow: Branching; Control Flow: Looping; Recursion; Writing function in Julia.


JU 203 Advanced Programming Concepts in Julia

In this section, we will be working with external files. The file either contains Julia code (module) or data(files). Furthermore, we will talk about some advanced concepts, such as exception and generic programming. This course will cover: Introduction to Modules in Julia; Introduction to Files in Julia; Structs and Objects; Exception Handling.

JU 204 Data Science in Julia

In this course, we will be working with the Julia libraries that are related to Data Science. Furthermore, we will introduce the mathematics that related to data science and their implementations. This course will cover: Introduction to ScikitLearn.jl; Data Visualization and Plotting using Plots.jl; Automatic differentiation and finance in Julia; Monte Carlo method using Julia; Data Science Challenges and Future in Julia.
